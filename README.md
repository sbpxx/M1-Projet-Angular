# M1-Projet-Angular

> Angular single page application — TypeScript / HTML / CSS — M1 university project.

## Description

**Single Page Application (SPA)** built with the **Angular** framework as part of an M1 course. The project leverages TypeScript, Angular's component system and modern front-end tooling.

## Tech stack

| Technology | % |
|---|---|
| TypeScript | 74.3% |
| HTML | 13.4% |
| CSS | 12.3% |

## Requirements

- [Node.js](https://nodejs.org/) 18+
- Angular CLI: `npm install -g @angular/cli`

## Install & start

```bash
cd M1-Projet-Angular
npm install
ng serve
```

Open [http://localhost:4200](http://localhost:4200) in your browser.

## Production build

```bash
ng build --configuration production
```

Compiled files are output to the `dist/` folder.
